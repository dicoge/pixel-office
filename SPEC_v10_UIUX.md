# Pixel Office v10 — UI/UX 美化（Manus 擔當）

## 目標
全面提升 Pixel Office 的視覺品質，讓介面看起來精緻、溫暖、專業。
Manus 負責**所有視覺美化**，不碰功能邏輯。

## 範圍

### 1. 室內場景美化（game.js drawRoom）
- **背景色調統一**：目前用 office_bg_small.webp（冬季山屋），色調偏冷藍。改成暖色系（奶油色牆面、木紋地板）
- **沙發更精緻**：紅色沙發加上陰影、枕頭細節、坐墊分層
- **書架更豐富**：書本顏色更多樣、加上小裝飾品（盆栽、相框、時鐘）
- **地板**：棋盤格地板加上木紋光澤感
- **照明**：加上暖色燈光 overlay（左上角有燈源感）
- **桌子**：4張小桌加上桌腳細節、桌面反光

### 2. 角色視覺提升（game.js placeCharacters）
- **角色陰影**：角色腳下加上小橢圓陰影（半透明黑），增加立體感
- **名稱標籤**：背景半透明黑底 + 白色粗體字，加上箭頭指向角色
- **狀態徽章**：徽章加上發光邊框或動畫光暈
- **Hermes 星星**：加上旋轉光暈特效（粒子或環形發光）
- **角色動畫過渡**：移動時加上簡易的彈跳/飄移效果

### 3. UI 介面美化（index.html / style.css）
- **頂部導航欄**：加上漸變背景、圓角、左側 logo + 右側按鈕
- **側邊部門面板**：半透明毛玻璃效果（backdrop-filter: blur）、圓角、平滑動畫
- **團隊成員列表**：加上頭像小圖示、狀態點點有脈動動畫
- **狀態文字區域**：打字機效果保持、加上打字游標閃爍
- **載入畫面**：加上動畫 logo（旋轉星星）、進度條更華麗
- **整體配色方案**：深色主題（#1a1a2e 背景）+ 金色 (#ffd700) 點綴 + 溫暖色調

### 4. 互動效果
- **按鈕 hover**：hover 時放大 + 發光邊框
- **角色點擊**：點擊角色會跳出名片/狀態卡（可選）
- **載入動畫**：頁面載入時角色從邊緣走進辦公室

## 修改檔案
- `/tmp/pixel-office/public/game.js` — 場景繪製、角色視覺、動畫效果
- `/tmp/pixel-office/public/index.html` — 整體佈局、樣式、互動效果
- `/tmp/pixel-office/public/style.css`（如果存在）— 獨立樣式

## 技術限制
- 遊戲引擎：Phaser 3.60 Canvas mode
- 場景解析度：1280x720
- 所有角色用 Sprite 物件（不是 Image/Rectangle）
- 不要改 MEMBERS、TOOL_COLORS、STATES、AREAS 的邏輯
- 不要改 API 整合（fetchStatus、loadMemo、loadDepartments）
- 不要改聊天/WebSocket 功能

## 驗收條件
1. 整體視覺溫暖一致，有「咖啡店辦公室」的氛圍
2. 角色有立體感（陰影 + 標籤 + 徽章動畫）
3. 介面元素精緻（毛玻璃、漸變、動畫）
4. 載入畫面有設計感
5. 沒有破版或 JS 錯誤
